# Mealgro Rider

A modern Flutter app built for delivery riders to manage their orders, navigate routes, and track deliveries easily on Android and iOS. 🚴📱

## What is Mealgro Rider?

Mealgro Rider empowers delivery personnel with real-time order updates, route guidance, and tools to efficiently complete meal deliveries. It’s the companion app designed specifically for riders, streamlining daily delivery operations.

## Project Structure

```
android/      # Android platform code
assets/       # Images & fonts
ios/          # iOS platform code
lib/          # Flutter source code
test/         # Unit & widget tests
pubspec.yaml  # Dependencies & project config
```

## Quick Start

1. **Clone:**  
   `git clone https://github.com/mellcky/Mealgro_Rider.git`
2. **Install:**  
   `flutter pub get`
3. **Run:**  
   `flutter run`
